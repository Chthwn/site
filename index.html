Этот простой сайт продемонстрирует вам пару алгоритмов шифрования

Атба́ш (ивр. אתבש‎) — простой шифр подстановки для алфавитного письма. Правило шифрования состоит в замене i-й буквы алфавита буквой с номером n − i + 1,

где n — число букв в алфавите.
